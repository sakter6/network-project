DNS

DNS (Domain Name System) converts a website name into an IP address.

When a user enters a website URL, DNS helps the browser find the correct server that hosts the website.

IP Addressing

An IP address identifies a device or server on the Internet. Websites can use two types of addresses:

IPv4 – example: 185.199.108.153
IPv6 – example: 2606:50c0:8000::153

Protocols

HTTP is the protocol used to transfer web pages between a browser and a server. HTTPS is the secure version of HTTP and encrypts the communication to protect data.
